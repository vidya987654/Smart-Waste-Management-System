🌿 Platform Features

📡

IoT Real-Time Monitoring

Ultrasonic sensors in each bin report fill levels every 15 minutes to the central dashboard via MQTT protocol.

🤖

AI Route Optimization

Greedy nearest-neighbour algorithm plans optimal garbage truck routes based on bin criticality and location.

🔔

Smart Alerts

Instant notifications when bins exceed 85% capacity — preventing overflow and maintaining city hygiene.

📊

Analytics Dashboard

Zone-wise fill analysis, weekly collection trends, and waste type breakdowns for smarter city planning.

🗺️

Interactive Map

Live Leaflet.js map showing all bin locations, fill levels, and optimized truck collection routes.

🔗

REST API

Full JSON API for IoT device integration, mobile apps, and third-party municipal system connectivity.
